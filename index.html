连接教学 / 安装指南

📘 教学说明

本文为《LoveLive! 学园偶像祭》连接教学及安装指南，分iOS与安卓两个客户端版本进行说明。

🍎 iOS 安装教程

需要提前准备的工具：

  • 电脑（Windows / macOS）
  • 登录游戏需要的苹果设备（iPhone / iPad）
1. 在电脑端点击图标，下载提供的 LLSIF.ipa 文件
2. 在电脑端前往下载 Sideloadly! ：
https://sideloadly.io/
3. 启动Sideloadly!，并将ipa文件拖入左侧方框中，在iDevice下拉框中找到需要游玩的苹果设备，点击开始 (Start) 按钮进行安装
4. 会有弹窗提示登录Apple ID（用于签名），输入自己的账号与密码即可
5. 运行途中确保苹果设备是解锁并亮屏的状态，等待下方的Log出现完成的字样后即安装成功，手机端会出现安装的应用
6. 在苹果设备上开启开发者模式：
设置 → 隐私与安全性 → 开发者模式。点击打开开发者模式后面的按钮。在下方弹出的窗口中，点击选择重启即可
7. 重启后，需要信任应用：
设置 → 通用 → VPN与设备管理，会找到下方开发者应用中自己刚才登录的Apple ID，是未信任状态，点击信任并输入解锁PIN，即可启动应用程序
8. 启动后，检查顶部的绿色字体为对应的URL: sif.mihane.cc ，即为成功，点击屏幕开始建号下载数据
⚠ 注意：此iOS安装方法为旁加载，签名时效七天，到期需重复步骤3-5，无需且请勿卸载应用，会导致账号丢失！

🤖 安卓安装教程

需要提前准备的工具：

  • 登录游戏需要的安卓设备一台
1. 点击图标，下载 APK 文件
2. 点击安装，根据自身定制系统的情况进行信任并安装应用
3. 安装完成后启动一次然后退出，注意不需要点击开始画面
4. 长按桌面上的应用，弹出服务器设置 (Server Config)的按钮，点击进入
5. 将URL修改为对应服务器地址：
http://sif.mihane.cc
6. 启动后，检查顶部的绿色字体为对应的URL: sif.mihane.cc ，即为成功，点击屏幕开始建号下载数据
⚠ 注意：请勿卸载应用，会导致账号丢失！
